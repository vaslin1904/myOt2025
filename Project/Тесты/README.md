
## Реализация

### 1. **Динамическая маршрутизация**
- Используется `frr` для OSPF между ВМ
- Все ВМ в одной внутренней сети `10.10.1.0/24`
- `dnsSrv` выступает в роли шлюза

### 2. **Репликация MySQL**
- GTID-репликация между `db-master` и `db-slave`
- Автоматическое восстановление при падении мастер-сервера
- `db-slave` может стать мастером в случае аварии

### 3. **Безопасность**
- `nftables` - брандмауэр для защиты
- `bind9` - DNS-сервер с ограничениями
- `angie` - reverse proxy с SSL
- Права доступа на файлы и директории

### 4. **Резервное копирование**
- Ежедневные бэкапы БД через `cron`
- Хранение в `/var/backups/mysql/`
- Скрипт автоматического резервного копирования

### 5. **Мониторинг**
- Prometheus + Grafana для сбора метрик
- Alertmanager для отправки уведомлений
- Мониторинг состояния БД, веб-серверов и репликации
